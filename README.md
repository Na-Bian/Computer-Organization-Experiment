## 实验内容

《计算机组成原理》课程实验材料，主要围绕运算器、存储系统以及 MIPS CPU 设计展开。实验内容覆盖 Logisim 电路仿真、MIPS 指令测试、单周期硬布线控制器 CPU 设计和多周期微程序控制器 CPU 设计等部分。

主要内容包括：

* 运算器实验
* 存储系统实验
* 单周期 MIPS CPU 设计
* 多周期 MIPS 微程序 CPU 设计

## 实验材料

`实验材料` 目录保留课程提供的电路文件、测试程序、自动生成表格、指令手册和部分演示视频。其中 CPU 设计实验部分包含单周期硬布线控制器和多周期微程序控制器相关材料，可用于复现实验设计与测试过程。

## 注意事项

* 仓库内容仅供课程学习和实验思路参考，请勿直接抄袭提交。
* 不同 Logisim 版本在存储器、ROM/RAM、探针工具和第三方库加载方面可能存在差异，复现实验时建议使用课程指定环境。
* 仓库中包含部分课程测试材料和演示视频，主要用于复现实验现象与报告截图来源。
